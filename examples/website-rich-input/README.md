# Website Rich Input example

Here we have an example Lexical setup in rich text configuration (`@lexical/rich-text`) with history (`@lexical/history`), hashtag (`@lexical/hashtag`) and custom character counter plugin

This example is used on the page https://lexical.dev/

**Run it locally:** `npm i && npm run dev`
