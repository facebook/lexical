# `@lexical/code`

[![See API Documentation](https://lexical.dev/img/see-api-documentation.svg)](https://lexical.dev/docs/api/modules/lexical_code)

This package contains the node and utility APIs for Lexical code blocks.

Prism-based syntax highlighting is provided by [`@lexical/code-prism`](https://www.npmjs.com/package/@lexical/code-prism).
