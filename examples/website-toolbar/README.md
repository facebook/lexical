# Website Toolbar example

This example demonstrates the implementation of a toolbar with a set of buttons that execute commands and set text formats for the selection. Buttons can be active depending on where the selection is set

This example is used on the page https://lexical.dev/

**Run it locally:** `pnpm i && pnpm run dev`
